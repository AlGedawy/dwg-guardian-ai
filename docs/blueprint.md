# **App Name**: DWG Guardian AI

## Core Features:

- AI Standard Auditor Tool: A standards-based CAD auditing tool that uses reasoning to evaluate drawing quality and detect incorrect layers, text inconsistencies, annotation issues, plotting risks, and scale problems across DWG, DXF, and PDF uploads.
- Automated Remediation Tool: A rule-driven remediation tool that uses reasoning to suggest precise CAD fixes for duplicate geometry, plotting risks, incorrect layers, lineweight problems, text inconsistencies, and scale violations.
- Health Scoring Visualizer: A technical dashboard providing a 0-100 drawing health score based on compliance metrics, featuring precision charts and priority-based warnings.
- Project Vault with Supabase: A centralized project management system for secure storage, versioning, and audit history of DWG, DXF, and PDF engineering files.
- Enterprise QA Reports: Professional engineering quality assurance system that generates detailed PDF reports including health scores, violation summaries, and audit histories.
- Interactive Analysis Workspace: A high-density responsive viewport for inspecting detected CAD errors, filtering issues by severity, and reviewing AI-driven cleanup suggestions.

## Style Guidelines:

- Deep charcoal background (#121212) with a subtle grid texture, electric blue accents (#007BFF), and specific status colors for warnings (amber) and errors (red).
- 'Space Grotesk' for technical headings and UI labels; 'Inter' for neutral, readable body text; 'Source Code Pro' for CAD attributes and coordinate data.
- Geometric thin-line icons inspired by CAD drafting tools and architectural drafting symbols for a professional engineering feel.
- Organized high-density grid system inspired by Linear, featuring a left sidebar navigation, technical cards, and engineering-style data panels.
- Ultra-fast micro-interactions including skeleton loading states, instant hover responses, and smooth panel transitions to emphasize tool-like efficiency.
